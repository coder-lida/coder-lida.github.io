RSA 加密原理

步骤 说明 描述 备注
1 找出质数 P 、Q -
2 计算公共模数 N = P * Q -
3 欧拉函数 φ(N) = (P-1)(Q-1) -
4 计算公钥E 1 < E < φ(N) E的取值必须是整数E 和 φ(N) 必须是互质数
5 计算私钥D E * D % φ(N) = 1 -
6 加密 C ＝ M E mod N C：密文 M：明文
7 解密 M ＝C D mod N C：密文 M：明文

公钥＝(E , N)
私钥＝(D, N)

对外，我们只暴露公钥。
